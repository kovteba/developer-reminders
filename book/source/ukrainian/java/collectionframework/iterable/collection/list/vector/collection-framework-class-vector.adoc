[#java-collection-framework-vector]
====== Vector
Реалізація динамічного масиву об'єктів. Vector з'явився в Java 1.0, але, як і `Hashtable`, не рекомендується використовувати цю колекцію, якщо не потрібно досягнення потокобезпеки. Оскільки в Vector, на відміну від інших реалізацій `List`, всі операції з даними є синхронізованими. Як альтернативу часто застосовується аналог - `ArrayList`. Але ця колекція відмічена як `deprecated` і використовується рідко.
