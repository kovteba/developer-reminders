=== Python

Python — це високорівнева, інтерпретована мова програмування з динамічною типізацією. Вона була створена Гвідо ван Россумом і вперше випущена в 1991 році. Python відомий своєю простотою та читабельністю, що робить його популярним вибором для початківців та професіоналів.

Всі оператори в Python є функціями, що дозволяє використовувати їх як об'єкти першого класу. Це означає, що ви можете передавати оператори як аргументи в інші функції, зберігати їх у змінних або структурах даних, і навіть створювати свої власні оператори:

- ** - Оператор піднесення до степеня (Exponentiation): Використовується для піднесення числа до певного степеня. Наприклад, `2 ** 3` поверне `8`, оскільки 2 піднесене до степеня 3 дорівнює 8.
- // - Оператор цілочисельного ділення (Floor Division): Використовується для ділення двох чисел з округленням вниз до найближчого цілого. Наприклад, `7 // 3` поверне `2`, оскільки 7 поділене на 3 дорівнює 2 з залишком 1, і округляється вниз до 2.
- % - Оператор взяття залишку (Modulus): Використовується для отримання залишку від ділення двох чисел. Наприклад, `7 % 3` поверне `1`, оскільки 7 поділене на 3 дорівнює 2 з залишком 1.
- @ - Оператор матричного множення (Matrix Multiplication): Використовується для множення двох матриць. Наприклад, якщо `A` і `B` є двома матрицями, то `A @ B` виконає матричне множення між ними.
- := - Оператор присвоения выражения (Walrus Operator): Позволяет присваивать значение переменной в рамках выражения. Например, `if (n := len(my_list)) > 5:` позволяет одновременно вычислить длину списка и проверить, больше ли она 5.
- | - Оператор побітового OR (Bitwise OR): Використовується для виконання побітової операції OR між двома числами. Наприклад, `5 | 3` поверне `7`, оскільки в двійковому представлении 5 (0101) и 3 (0011) дают 7 (0111).
- & - Оператор побітового AND (Bitwise AND): Використовується для виконання побітової операції AND між двома числами. Наприклад, `5 & 3` поверне `1`, оскільки в двійковому представлении 5 (0101) и 3 (0011) дают 1 (0001).
- ^ - Оператор побітового XOR (Bitwise XOR): Використовується для виконання побітової операции XOR между двумя числами. Например, `5 ^ 3` вернет `6`, поскольку в двоичном представлении 5 (0101) и 3 (0011) дают 6 (0110).
- / - Оператор ділення (Division): Використовується для ділення двох чисел. Наприклад, `7 / 3` поверне `2.3333333333333335`, оскільки 7 поділене на 3 дорівнює приблизительно 2.33.
